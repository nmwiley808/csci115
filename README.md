## Course Description

CSCI 115 focuses on the design, implementation, and analysis of **algorithms and data structures**, which are essential for efficient software development.

Topics covered in this course include:

- Review of basic data structures  
- Arrays, linked lists, stacks, and queues  
- Trees and binary search trees  
- Graph structures and graph traversal algorithms  
- Shortest path algorithms  
- Spanning trees  
- Sorting algorithms (merge sort, quicksort, heap sort, etc.)  
- Searching algorithms  
- Algorithm design and analysis  
- Memory management  
- Hashing techniques  
- Dynamic storage allocation  
- Integration of data structures into system design  

The course combines **theoretical analysis with practical implementation** of data structures and algorithms.
